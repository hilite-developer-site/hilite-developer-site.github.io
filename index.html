Welcome!

This site intends to be a living set of documents containing all the information necessary to integrate HILITE into your systems and applications. Contact the team should you find a problem or have questions about how to use this information. Much more coming soon!

Getting Started

  1. Before you can access the HILITE api, you'll need to get your token. Check our authentication docs to find out how.
  2. Once you can access you'll want to create and manage a campaign. View our campaign docs for more details
  3. Once you have one or more campaigns created, you'll want to learn how to upload videos to them. View our deliverable docs to find out more information.
